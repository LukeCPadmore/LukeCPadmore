## Featured Personal Projects

### Deep Learning Image Colourisation

End-to-end image colourisation system built from research papers, trained from scratch in PyTorch and deployed as a production-style demo on Hugging Face Spaces.

**Tech:** PyTorch, FastAPI, ONNX Runtime, Docker, MLflow, Hugging Face Spaces

* Implemented and trained a PyTorch image colourisation model from scratch
* Served the model through a FastAPI endpoint using ONNX Runtime for optimised inference
* Used MLflow to track configs, checkpoints, metrics and generated image samples

Links: [Repo](https://github.com/LukeCPadmore/flow-matching) | [Demo on HF Spaces](https://huggingface.co/spaces/LukeCPadmore/colouriser-infra) (Under development)

### PyTorch Hyperparameter Tuning CLI

Python CLI tool for automating Optuna hyperparameter optimisation for PyTorch Lightning models.

**Tech:** Python, Typer, uv, Optuna, PyTorch Lightning, GitHub Actions, PyPI

* Built a reusable CLI for configurable and reproducible HPO workflows
* Added CI/CD with GitHub Actions for testing, package builds and PyPI publishing
* Designed the tool to reduce repeated experiment setup code for ML projects

Links: [Repo](https://github.com/LukeCPadmore/optuna-lightning-cli) | [![PyPI](https://img.shields.io/pypi/v/optuna-lightning-cli.svg)](https://pypi.org/project/optuna-lightning-cli/)
